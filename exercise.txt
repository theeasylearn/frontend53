**Section Name: CSS Selectors**

**Easy Exercise 1**  
* **Title:** Basic Element Selector for Navigation Menu  
* **Difficulty Level:** Easy  
* **Problem Statement:** Create a simple navigation bar with links (Home, About, Services, Contact). Use element selectors to style all `<a>` tags with blue text, no underline, and padding.  
* **Expected CSS Concepts Used:** Element (tag) selectors, basic properties like color, text-decoration, padding.  
* **Output Requirements:** Horizontal navigation bar with evenly spaced links that change color on hover (use a:hover). Provide basic HTML structure.

**Easy Exercise 2**  
* **Title:** Class Selector for Highlighted Boxes  
* **Difficulty Level:** Easy  
* **Problem Statement:** Create three product cards. Apply a class "highlight" to one card and use class selector to give it a yellow background and bold text.  
* **Expected CSS Concepts Used:** Class selectors (.), background-color, font-weight.  
* **Output Requirements:** Visually distinct highlighted card among plain ones.

**Medium Exercise 1**  
* **Title:** ID Selector for Hero Banner  
* **Difficulty Level:** Medium  
* **Problem Statement:** Build a hero section with a heading and button. Use an ID selector for the hero container to center content and add a background color. Style the button using a class.  
* **Expected CSS Concepts Used:** ID selectors (#), text-align, margin/padding for centering.  
* **Output Requirements:** Centered hero with prominent button.

**Medium Exercise 2**  
* **Title:** Combinator Selectors for Article Styling  
* **Difficulty Level:** Medium  
* **Problem Statement:** Create an article with paragraphs and a nested list. Use descendant and child selectors to style paragraphs inside the article differently from others and list items.  
* **Expected CSS Concepts Used:** Descendant (space), child (>), sibling selectors.  
* **Output Requirements:** Differentiated text styles within the article structure.

**Hard Exercise**  
* **Title:** Advanced Selectors for Interactive Form  
* **Difficulty Level:** Hard  
* **Problem Statement:** Design a contact form with labels, inputs, and a submit button. Use attribute selectors for required fields, pseudo-classes for focus states, and group selectors for consistent styling. Combine with previous concepts like classes.  
* **Expected CSS Concepts Used:** Attribute selectors ([ ]), pseudo-classes (:focus, :hover), grouping, descendant selectors.  
* **Output Requirements:** Form with visual feedback on focus/required fields; clean, professional look.  
* **Bonus Challenge:** Add :nth-child selectors to alternate row colors if extending to a table of past messages.
